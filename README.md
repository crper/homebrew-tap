# crper/homebrew-tap

Homebrew tap for `crper` command-line tools.

## Install

```bash
brew tap crper/tap
brew install --cask crper/tap/tqrx
```

Or install directly:

```bash
brew install --cask crper/tap/tqrx
```

## Notes

- Casks in this repository are generated and updated by GoReleaser.
- The main source repository for `tqrx` is <https://github.com/crper/tqrx>.
- Generated casks live in the `Casks/` directory.
